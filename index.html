Technical Overview

RudeAPI is a secure, anonymous middleware for developer utilities. No registration, no tracking, just pure data endpoints served via HTTPS. Built with Python 3.9+ and FastAPI.

API Endpoints

GET

/tools/password

Generates a secure random alphanumeric password. Perfect for user onboarding or temporary credentials.

Parameters

length Optional 8 to 64 (Default: 16)

JSON Response

{ "password": "x7P$kQ29!vLm5RzW8uT1", "length": 20 }
GET

/tools/id

Returns a globally unique 16-character hexadecimal identifier.

Parameters

No parameters required.

{ "unique_id": "ID-AABBCCDDEEFF0011" }
GET

/tools/hash

Converts a string into a SHA256 hash. Purely computational, no data is logged.

Parameters

text Required Any UTF-8 string

Frequently Asked Questions

Is there an API key required?

No. RudeAPI is open and anonymous. We believe in unrestricted utility for developers.

What is the rate limit?

We allow up to 120 requests per minute per origin. If you exceed this, you'll receive a 429 Error.

Why "RudeAPI"?

Because it's direct, doesn't waste time with fluff, and gives you blunt answers in the Magic 8-ball tool.
